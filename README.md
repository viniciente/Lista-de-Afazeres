# 📋 Lista de Afazeres

Uma aplicação simples e prática para gerenciar suas tarefas diárias. Organize suas atividades, marque as concluídas e nunca mais esqueça do que precisa fazer!

## 🎯 O que é?

Este projeto é um **gerenciador de tarefas (To-Do List)** desenvolvido em Pascal. Ele permite que você:

- ✅ Adicione novas tarefas
- ✅ Marque tarefas como concluídas
- ✅ Remova tarefas da lista
- ✅ Visualize todas as suas atividades organizadas

## 🚀 Como Usar

### Pré-requisitos

Para rodar este projeto, você precisa ter:
- **Pascal** instalado em seu computador (Free Pascal ou Turbo Pascal)
- Um compilador Pascal configurado

### Instalação

1. **Clone o repositório:**
```bash
git clone https://github.com/viniciente/Lista-de-Afazeres.git
cd Lista-de-Afazeres
```

2. **Compile o programa:**
```bash
fpc seu_arquivo.pas
```
> *Substitua `seu_arquivo.pas` pelo nome do arquivo principal do projeto*

3. **Execute a aplicação:**
```bash
./seu_arquivo
```

## 📖 Funcionalidades Principais

### 1. Adicionar Tarefa
- Digite a nova tarefa que deseja adicionar
- A tarefa será salva automaticamente na lista

### 2. Visualizar Lista
- Veja todas as suas tarefas em um só lugar
- Cada tarefa é numerada para facilitar a identificação

### 3. Marcar como Concluída
- Selecione a tarefa que foi realizada
- A tarefa será marcada como ✓ concluída

### 4. Remover Tarefa
- Escolha a tarefa que deseja deletar
- A tarefa será removida da lista permanentemente

## 💡 Dicas de Uso

- **Seja específico:** Escreva tarefas claras e detalhadas
- **Priorize:** Coloque as tarefas mais urgentes no topo
- **Revise regularmente:** Confira sua lista diariamente
- **Comemore as vitórias:** Marque como concluída cada tarefa finalizada!
